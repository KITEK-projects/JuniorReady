# Junior Ready

Веб-приложение Junior Ready разработано как единостраничное приложение (Single Page Application) на основе библиотеки React. Выбор технологического стека обусловлен несколькими факторами: React является наиболее востребованным фронтенд-фреймворком на рынке труда в исследуемой категории вакансий; приложение демонстрирует студентам реальный пример применения изучаемых технологий; архитектура SPA обеспечивает работу приложения без серверной части, что упрощает деплой и обслуживание.
